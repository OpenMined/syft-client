# syft-bg

Background services TUI dashboard for SyftBox.

## Installation

```bash
pip install syft-bg
```

## Usage

```bash
# Show status of all services
syft-bg status

# Start all services
syft-bg start

# Stop all services
syft-bg stop

# Launch TUI dashboard
syft-bg
```

## Services

- **notify** - Email notifications for job and peer events
- **approve** - Auto-approve jobs and peer requests
